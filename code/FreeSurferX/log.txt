1. check_env.sh: check FreeSurfer environment
2. run_recon.sh: run default recon-all pipeline
3. extract_TIV.sh: extract TIV measure
4. extract_Euler.sh: extract Euler number
5. get_tissue_mask.sh: extract tissue masks
6. run_surface_parc.sh: parcellate cortex using external atlases
7. apply_builtin_parc.sh: apply builtin parcellations
8. calc_LGI.sh: calculate LGI feature
9. extract_cortical_feature.sh: extract cortical features
11. ZR/clean_cortical_feature.R: extract cortical features in final form
12. extract_subcort_volume.sh: extract subcortical volume feature
13. ZR/clean_subcort_volume.R: extract subcortical volume in final form
14. get_parc_mask.sh: extract parcellation masks
15. ZR/clean_parc_mask.R: extract parcellation masks in final form
16. plot_volume_parc.sh: make a figure of volume parcellation for quality check
17. plot_surface_parc.sh: make a figure of surface parcellation for quality check
